input n- ввод данных

output n- вывод данных

set n val - положить в ячейку с номером n значение val

add n1 n2 n3 - сложить ячейки с номерами n1 и n2, результат в n3

sub n1 n2 n3 - вычесть из ячейки с номером n1 ячейку n2, результат в n3

mul n1 n2 n3 - умножить ячейку n1 на ячейку n2, результат в n3

del n1 n2 n3 - разделить ячейку n1 на ячейку n2, результат в n3


команды, результат которых (0 или 1):

cmp n1 n2 n3 - проверяет, равны ли значения ячеек с номерами n1 и n2, результат ячейку n3

mti n1 n2 n3 - проверяет больше ли значение ячейки n1 ячейки n2, результат ячейку n3

lti n1 n2 n3 - проверяет меньше ли значение ячейки n1 ячейки n2, результат ячейку n3

meti n1 n2 n3 - проверяет больше или равно ли значение ячейки n1 ячейки n2, результат ячейку n3

leti n1 n2 n3 - проверяет меньше или равно ли значение ячейки n1 ячейки n2, результат ячейку n3

jmp n - переместиться на ячейку с номером n

jz n1 n2 - если в ячейке с номером n1 находится 0, то переместиться на ячейку с номером n2

jnz n1 n2 - если в ячейке с номером n1 находится 1, то переместиться на ячейку с номером n2


exit - выход из программы (если программа не содержит этой команды, она завершится аварийно)
